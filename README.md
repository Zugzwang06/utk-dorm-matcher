Dorm Matcher

A web application built with React that helps incoming University of Tennessee students find dorms that best match their preferences.

Students answer a series of questions about what matters most to them, such as location, cost, room style, and campus lifestyle. Based on their responses, the application recommends dorms that may be a good fit.

While most students eventually learn which dorms are popular, this project was built as a way to organize dorm information into a simple recommendation system and provide an easier starting point for students unfamiliar with campus housing options.

Why I Built It

I wanted a project that would let me practice React in a more creative way than a standard tutorial application. Instead of building something solely to demonstrate React skills, I chose a topic that was familiar to students at UTK and used it as an opportunity to design a recommendation-based application.
